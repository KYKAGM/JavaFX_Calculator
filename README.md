# JavaFX_Calculator
Калькулятор на JavaFX с интерфейсом в стиле iOS, созданный через SceneBuilder и построенный по архитектуре MVC. Поддерживает основные операции, кастомные элементы интерфейса и пасхалки: при получении результатов 1889 или 67 воспроизводятся звуки. Проект полностью рабочий и легко расширяемый.
